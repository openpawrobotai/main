## 🚀 Quick Start

```bash
# Clone the firmware repository
git clone https://github.com/ayvalabs/openpaw-firmware
cd openpaw-firmware

# Set target and build
idf.py set-target esp32
idf.py build

# Flash to device and monitor
idf.py flash monitor
```
